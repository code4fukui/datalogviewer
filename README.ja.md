# データログビューワー

このプロジェクトは、圧力やその他のセンサーデータをグラフ表示するWebアプリケーションです。CSVファイルをドロップすると、自動的にラインチャートに描画されます。

## 機能
- CSVファイルをドロップするだけで自動的にグラフ表示
- 複数のデータ系列をラインチャートで表示
- 時間軸にはタイムスタンプが表示され、わかりやすい

## 必要環境
特にありません。ウェブブラウザがあれば利用できます。

## 使い方
1. CSVファイルをドロップする
2. ファイル内のタイムスタンプとデータ列が自動的にグラフ化される

## データ/API
プロジェクトには、模擬的な圧力と相対高度データを含む `Barolog-2025-09-10.csv` サンプルファイルが含まれています。
